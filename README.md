# 🍔 Online Food Delivery SQL System

This project is a SQL-based database system that simulates a real-world food delivery platform.

## 📊 Features
- Customer management
- Restaurant and menu system
- Order processing
- Payment tracking
- Delivery management

## 🧠 SQL Concepts Used
- Joins (INNER, LEFT)
- Group By & Aggregations
- Subqueries
- ENUM
- Foreign Keys

## 📈 Analysis Performed
- Top customers by orders
- Revenue calculation
- Most ordered items
- Monthly revenue trends
- Customers with no orders

## 🛠️ Tech Used
- MySQL

## 🚀 Project Structure
- schema.sql → database design
- data.sql → dummy data
- queries.sql → analysis queries
